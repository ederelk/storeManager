简介

JComponentPack 是一个基于Java Swing的GUI类库，一系列可视化的JavaBeans集合，它基于Swing的MVC架构，是100%的纯Java类库，它包括20多个Swing 所没有的控件，如EXCEL风格Java的电子表格控件，读取数据库表的Java表格控件，日历控件，类似资源管理器的列表控件，向导框架等等，这些 JavaBeans可以直接拖动到您的图形用户界面，它可提高您的工作效率，加快您的项目开发过程，给您的团队节省时间，减少开支。

系统要求

Java 运行环境 1.5.0 或更高版本.

JComponentPack的内容

本部分介绍了JComponentPack中的文件和目录。
类库
(位于 lib/ 子目录中) JComponentPack标准的swing类库和可选包.
插件
(位于 plugins/ 子目录中) 为NetBeans IDE开发的JComponentPack插件, 也包含了一个免费的为NetBeans IDE开发的合并属性文件插件.
Web类库
(位于 web/ 子目录中) JComponentPack产品的Web类库及演示程序.
文档 - 需要JComponentPack二进制授权, 现在订购!
(位于 docs/ 子目录中) JComponentPack产品的程序员指南和Javadoc API文档.
桌面演示程序源代码 - 需要JComponentPack二进制授权, 现在订购!
(位于 demosrc/ 子目录中) JComponentPack产品的演示程序源代码.
桌面swing控件源代码 - 需要JComponentPack源代码授权, 现在订购!
(位于 src/ 子目录中) JComponentPack产品的swing控件源代码.
Web类库源代码 - 需要JComponentPack源代码授权, 现在订购!
(位于 websrc/ 子目录中) JComponentPack产品的Web类库源代码.

再分发

这两个文件componentpack.jar和license.jar是在您的桌面应用程序或者Applet中部署我们的产品JComponentPack所必须的类库, 其它类库是可选的, 适合在Applet环境中使用“lazy”分发，JComponentPack产品的核心swing类库非常小，尤其在使用pack200工具压缩后。

这两个文件componentpack-web.jar和license.jar是在您的Web应用程序中部署我们的产品JComponentPack所必须的类库。
