JDataGrid Spreadsheet Web Components

The JDataGrid Spreadsheet Web Components is a new innovation in JComponentPack 3.0, it's different with the traditional Applet-Servlet communication components, all configuration can be completed on the server, the JDataGrid spreadsheet clients includes JDataGrid spreadsheet Applet only need specify the servlet URI, then the license will setup automatically, open and save the spreadsheet file will work magically.

It has the following features:
  • Read and configure the license data from web server automatically.
  • Open spreadsheet file from web server in background thread magically.
  • Save spreadsheet file to web server in background thread magically.
  • Configure the server's directory to read, save the spreadsheet file.
  • Provide API to hook the server operation includes read and save the spreadsheet file.

JDataTable Web Components

The JDataTable Web Components is a new innovation in JComponentPack 3.0, all configuration can be completed on the server, the JDataGrid database clients includes JDataGrid database Applet only need specify the servlet URI, then the license will setup automatically, it will load the appropriate JDBC driver automatically, read the RecordSet from the Database, save the RecordSet changes to the database magically.

It has the following features:
  • Read and configure the license data from web server automatically.
  • Load the appropriate JDBC driver automatically.
  • Read the RecordSet from the Database magically.
  • Save the RecordSet changes to the database magically.
  • Provide API to hook the server operation includes read and save the record set.
